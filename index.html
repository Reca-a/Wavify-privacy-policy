Wavify - Privacy Policy

Last updated: 2025-12-15

This application ("App") is developed as part of an educational project for academic purposes by Adrian Raczek. I respect your privacy and am committed to protecting it.

1. Data Collection

The App does not collect, store, or share any personal data from users.

The App operates fully offline and does not require user accounts, login, or internet-based data processing.

2. Device and App Information

The App may access basic device information provided by the Android operating system (such as Android version or device model) solely for proper functionality. This information is not stored, transmitted, or used to identify users.

3. Third-Party Libraries

The App uses standard Android and open-source libraries, including:

  • AndroidX libraries
  • Media3 / ExoPlayer (for media playback)
  • Coil (for image loading)
  • Material Components

These libraries do not collect personal data on behalf of the App.

4. App Permissions

The App requires the following permissions to function properly:

  • Access to media files (audio) - used only to allow the App to read and play audio files stored locally on the user's device. The App does not upload, store, or share these files.
  • Notifications - used to display playback controls while audio is playing in the background.

All permissions are used solely for core application functionality.

5. Data Security

Since no personal data is collected or stored, there is no risk of personal data leakage. The App follows standard Android security practices.

6. User Rights

As the App does not collect personal data, there is no personal data to access, modify, or delete.

7. Changes to This Privacy Policy

This Privacy Policy may be updated if the App functionality changes. Any updates will be published on this page.
